crs-binding examples application.
This application uses two main components from crs

  • crs-binding
  • crs-router

All navigation is handled by crs-router.
All binding is handled by crs-binding.

The purpose of this application is to document features of crs-binding using examples of how you can use it. These examples are not meant to be the end all and be all of web development. They are short little samples demonstrating how you could achieve certain things.
